# 王者荣耀自动识别点击工具

版本：0.0.1

这个工具通过模拟器 ADB 截图识别页面，再通过 ADB 点击指定坐标。它不会读写游戏目录、账号数据或游戏内存。

## 桌面版使用

双击：

```text
start_desktop.bat
```

桌面版提供这些按钮：

- `刷新设备`：检查 ADB 设备。
- `自动检测`：自动查找 ADB、已连接设备和游戏画面 display，并写入 `config.yaml`。
- `开始循环`：持续识别并按配置执行。
- `停止`：停止循环。
- `保存模板`：把当前屏幕保存成模板，方便游戏更新后重新学习页面。

界面左上角的 `真实点击` 默认不勾选。只有勾选后，程序才会真的通过 ADB 点击游戏。

## 推荐流程

1. 启动模拟器和王者荣耀。
2. 打开桌面版工具。
3. 点 `自动检测`，确认设备显示为已连接。
4. 先不勾选 `真实点击`，点 `开始循环` 看日志是否能识别当前页面。
5. 确认日志正常后，勾选 `真实点击`，再点 `开始循环`。
6. 要停止时点 `停止`。

## 配置说明

`config.yaml` 里主要看这几项：

```yaml
adb_path: ""
device: ""
display: "2"
interval: 1.2
```

- `adb_path`：模拟器自带 `adb.exe` 路径。留空时点 `自动检测` 会尝试自动填写。
- `device`：ADB 设备名。点 `自动检测` 会优先选择已连接设备。
- `display`：游戏画面所在 display。腾讯手游助手常见是 `2`，其他模拟器可能是 `0`。
- `interval`：循环识别间隔，单位秒。

如果换电脑、换模拟器或换安装路径，优先点 `自动检测`。

## 打包

第一次打包前运行：

```text
install_build_deps.bat
```

然后运行：

```text
build_exe.bat
```

打包完成后会生成：

```text
dist\wz_auto_tool\wz_auto_tool.exe
```

要发到别的 Windows 电脑，复制整个文件夹：

```text
dist\wz_auto_tool
```

也可以运行：

```text
make_portable_zip.bat
```

生成：

```text
wz_auto_tool_portable.zip
```

## 模板

发布包不会带明文 `templates` 文件夹，模板会被压缩到：

```text
templates.dat
```

如果游戏界面更新导致识别失效，可以在桌面版里用 `保存模板` 重新保存对应页面模板。

## 注意

- 程序只做菜单、匹配、选英雄和结算页面的识别点击。
- 游戏中状态只等待，不操作角色。
- 不同模拟器、分辨率、DPI 或游戏 UI 布局会影响识别效果。
