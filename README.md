# 王者荣耀自动练级工具

版本：0.0.4

这个工具通过模拟器 ADB 截图识别页面，再通过 ADB 点击指定坐标。它不会读写游戏目录、账号数据或游戏内存。

## 桌面版使用

双击：

```text
start_desktop.bat
```

桌面版提供这些按钮：

- `刷新设备`：检查 ADB 设备。
- `自动检测`：自动查找 ADB、已连接设备和游戏画面 display，并写入 `config.yaml`。
- `开始循环`：持续识别并按配置执行。
- `停止`：停止循环。

桌面版默认真实点击。点 `开始循环` 后会直接通过 ADB 点击游戏界面。

## 推荐流程

1. 启动模拟器和王者荣耀。
2. 打开桌面版工具。
3. 点 `自动检测`，确认设备显示为已连接。
4. 点 `开始循环` 开始自动识别和点击。
5. 要停止时点 `停止`。

## 配置说明

`config.yaml` 里主要看这几项：

```yaml
adb_path: ""
device: ""
display: "2"
interval: 1.2
```

- `adb_path`：模拟器自带 `adb.exe` 路径。留空时点 `自动检测` 会尝试自动填写。
- `device`：ADB 设备名。点 `自动检测` 会优先选择已连接设备。
- `display`：游戏画面所在 display。腾讯手游助手常见是 `2`，其他模拟器可能是 `0`。
- `interval`：循环识别间隔，单位秒。推荐 `1.0` 到 `1.5` 秒。

如果换电脑、换模拟器或换安装路径，优先点 `自动检测`。

## 打包

第一次打包前运行：

```text
install_build_deps.bat
```

然后运行：

```text
build_exe.bat
```

打包完成后会生成：

```text
dist\王者荣耀自动练级工具\王者荣耀自动练级工具.exe
```

要发到别的 Windows 电脑，复制整个文件夹：

```text
dist\王者荣耀自动练级工具
```

也可以运行：

```text
make_portable_zip.bat
```

生成：

```text
王者荣耀自动练级工具_v0.0.4.zip
```

## 模板

发布包不会带明文 `templates` 文件夹，模板会被压缩到：

```text
templates.dat
```

如果游戏界面更新导致识别失效，需要重新制作模板包后再打包发布。

## 注意

- 程序只做菜单、匹配、选英雄和结算页面的识别点击。
- 游戏中状态只等待，不操作角色。
- 不同模拟器、分辨率、DPI 或游戏 UI 布局会影响识别效果。
