# 王者荣耀自动练级工具

版本：1.0.1

这个工具通过模拟器 ADB 截图识别页面，再通过 ADB 点击指定位置。它只操作界面，不读取或修改游戏数据、账号数据或游戏内存。

## 桌面版使用

双击运行：

```text
start_desktop.bat
```

或在打包后的目录中双击：

```text
王者荣耀自动练级工具.exe
```

## 按钮说明

- `刷新设备`：重新检查当前可用的 ADB 设备。
- `自动检测`：自动查找 ADB、扫描常见模拟器端口、选择设备并检测游戏画面 display。
- `实时预览`：打开一个独立窗口查看当前 ADB 截图画面。
- `开始循环`：持续识别页面并按规则自动点击。
- `停止`：停止循环。

## 推荐设置

识别间隔推荐 `1.0` 到 `1.5` 秒。间隔越小，截图和识别越频繁；间隔越大，响应会更慢。

## 换电脑或换模拟器

第一次在新设备上使用时：

1. 启动模拟器。
2. 打开模拟器的 ADB 调试或开发者模式。
3. 打开游戏。
4. 启动本工具。
5. 点击 `自动检测`。
6. 显示已连接后，点击 `开始循环`。

腾讯手游助手、MuMu、雷电等模拟器都依赖 ADB 可用性。不同模拟器的端口和 display 可能不同，优先使用 `自动检测`。

## 配置

`config.yaml` 中常用配置：

```yaml
adb_path: ""
device: ""
display: "2"
interval: 1.2
```

- `adb_path`：模拟器自带 `adb.exe` 路径，留空时会自动查找。
- `device`：ADB 设备名，留空时自动选择已连接设备。
- `display`：游戏画面所在 display，腾讯手游助手常见是 `2`，其他模拟器可能是 `0`。
- `interval`：识别间隔，单位秒。

## 打包

第一次打包前运行：

```text
install_build_deps.bat
```

然后运行：

```text
build_exe.bat
```

生成便携压缩包：

```text
make_portable_zip.bat
```

输出：

```text
王者荣耀自动练级工具_v1.0.1.zip
```

## 注意

- 游戏中状态只等待，不操作角色。
- 如果游戏 UI、分辨率、DPI 或模拟器画面比例变化较大，识别模板可能需要重新制作。
- 发布包不带明文 `templates` 文件夹，识别模板已打包到 `templates.dat`。
