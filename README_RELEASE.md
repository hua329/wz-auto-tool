# 王者荣耀自动练级工具

版本：0.0.6

## 使用前准备

1. 安装并启动模拟器。
2. 在模拟器设置里打开 `开发者模式 / ADB 调试`。
3. 打开王者荣耀，并尽量保持模拟器分辨率和界面布局与打包时一致。
4. 双击运行 `王者荣耀自动练级工具.exe`。
5. 第一次在新设备或新模拟器上使用时，先点 `自动检测`。

## 按钮说明

- `刷新设备`：检查模拟器 ADB 是否连接。
- `自动检测`：自动查找 ADB、设备名和游戏画面 display，并写入配置。
- `开始循环`：持续识别并按规则自动点击。
- `停止`：停止循环。

## 使用方式

点 `开始循环` 后会默认真实点击。

要停止自动识别，点 `停止`。

## 注意

- 程序只通过 ADB 截图和 ADB 点击操作界面，不修改游戏数据。
- 游戏界面、分辨率、模拟器版本变化后，模板识别可能需要重新学习。
- 如果设备显示未连接，先确认模拟器已开启 ADB 调试，再点 `刷新设备`。程序会自动尝试连接常见本地模拟器 ADB 端口。
- 如果换了模拟器或安装路径，先点 `自动检测`。
- `间隔秒` 推荐设置为 `1.0` 到 `1.5` 秒，太小会更频繁截图，太大反应会慢。
