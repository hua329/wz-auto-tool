# 王者荣耀自动练级工具

版本：1.0.1

## 使用前准备

1. 启动模拟器。
2. 在模拟器设置中打开开发者模式或 ADB 调试。
3. 打开王者荣耀。
4. 双击运行 `王者荣耀自动练级工具.exe`。
5. 第一次在新设备或新模拟器上使用时，先点击 `自动检测`。

## 按钮说明

- `刷新设备`：重新检查模拟器 ADB 是否连接。
- `自动检测`：自动查找 ADB、扫描本机常见模拟器端口、检测设备名和游戏画面 display。
- `实时预览`：弹出实时游戏页面预览窗口。
- `开始循环`：持续识别并自动点击。
- `停止`：停止循环。

## 使用方式

点击 `开始循环` 后会真实点击游戏界面。

要停止自动识别，点击 `停止`。

## 推荐设置

识别间隔推荐 `1.0` 到 `1.5` 秒。太小会更频繁截图，太大会响应变慢。

## 注意

- 程序只通过 ADB 截图和 ADB 点击操作界面，不修改游戏数据。
- 游戏中只等待，不操作角色。
- 换模拟器、换电脑、换安装路径后，优先点击 `自动检测`。
- 如果设备显示未连接，请确认模拟器已开启 ADB 调试，再点击 `刷新设备` 或 `自动检测`。
