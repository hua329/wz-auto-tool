# Release Notes

## 0.0.3

- 自动检测和开始循环前会自动尝试连接常见本地模拟器 ADB 端口。
- 修复腾讯手游助手实际设备为 `127.0.0.1:5555` 时，旧配置 `emulator-5554` 导致未连接的问题。
- 刷新设备会在发现新在线设备后自动写回 `config.yaml`。

## 0.0.2

- 桌面版移除 `真实点击` 勾选项，默认真实点击。
- 游戏中保护规则改为左上角小地图模板匹配，避免人机模式等菜单页误判为游戏中。

## 0.0.1

`王者荣耀自动练级工具` 首个桌面版发布。

- 支持通过 ADB 自动检测模拟器设备和游戏画面 display。
- 支持桌面版循环识别并点击菜单、匹配、选英雄和结算页面。
- 游戏中优先通过左上角小地图识别并等待，降低对局内误点。
- 模板以 `templates.dat` 形式随程序发布，不直接暴露模板图片文件夹。
- 程序只使用 ADB 截图和 ADB 点击，不修改游戏数据、账号数据或游戏内存。
